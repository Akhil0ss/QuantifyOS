# 🚀 Quantify OS: Quick Start Guide

Welcome to the future of autonomous operations. This guide will help you get started with Quantify OS in under 5 minutes.

## 1. Login
Access the Quantify OS portal via your designated URL (typically `http://localhost:3000` for local setups).
- **Google/GitHub**: Click the respective icon for instant, secure authentication.
- **Email**: Enter your credentials and click "Continue with Email".
- **First Login**: A dedicated workspace and memory engine will be automatically provisioned for you.

## 2. Your First Task
The **Command Center** is your primary interface. It features a single, powerful input box.
1. Scroll to the "Command Center" tab.
2. In the box that says "Tell Quantify OS what to do...", type a simple instruction.
   - *Example*: "Research the latest trends in renewable energy and summarize them in a 3-bullet list."
3. Press **Enter** or click **INITIALIZE**.

## 3. Monitoring Execution
Once a task is submitted:
- The **Active Operations** feed will show the task's progress in real-time.
- You can click on the task to see a **Trace View** of exactly what the AI is thinking and doing (e.g., searching the web, writing files).

## 4. Basic Commands
Quantify OS understands natural language. Here are some command patterns you can use:
- **Research**: "Analyze [Topic] and save the results to research.txt"
- **Automation**: "Create a script to organize my downloads folder by file type."
- **Data**: "Read data.csv and find the average sales for Q3."
- **System**: "Check system health" or "Show my active capabilities."

## 5. Getting Help
If you're unsure what the system can do, look at the **Capability Index** on the right sidebar. It lists every tool the AI has currently "learned" and can use to help you.
